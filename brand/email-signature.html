How to add this signature to Gmail

  1. Select everything below the dashed line and copy it (Ctrl+A then Ctrl+C, or Cmd+A then Cmd+C on Mac — but only select the signature table, not this box)
  2. Open Gmail → click the gear icon → See all settings
  3. Stay on the General tab, scroll to Signature
  4. Click Create new, name it (e.g. "Professional")
  5. Click inside the signature text area and paste (Ctrl+V / Cmd+V)
  6. Scroll down and click Save Changes
